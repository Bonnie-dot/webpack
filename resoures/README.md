# dataanalysis

> A Vue.js project

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```
css->sass
js->es6
ajax->axios
打包注意 :
- 去调main.js 里面 api
- 去掉App.vue src的api
